Globale Befehle

Diese Befehle (die mit / beginnen), werden ausgeführt unabhängig vom Kanal in dem man sich befindet und anderen Benutzern nicht angezeigt.

Registrierung

Syntax: /register NAME PASSWORT MAILADRESSE

Aktuell gibt es keine Möglichkeit, sein Passwort zu ändern oder sich ein neues zuschicken zu lassen. Entsprechend: Gut merken!

Login

Syntax: /login NAME PASSWORT

ChanServ-Befehle

Diese Befehle (die mit ! beginnen), werden nur in Kanälen ausgeführt in denen sich ChanServ befindet und dann abhängig vom Kanal ausgeführt.

Zu einem Channel wird gespeichert, wer der Besitzer ist (Owner), außerdem Listen von Ops und Voices. Es findet keine Verwaltung anhand von Levels o.Ä. statt.

Benutzerliste anzeigen

Syntax: !users

Benötigte Rechte: Keine

Zeigt die von ChanServ verwaltete Liste von Benutzern an.

Benutzerliste verändern

Syntax: !addop BENUTZER|*ACCOUNT (In die Op-Liste eintragen)
Syntax: !addvoice BENUTZER|*ACCOUNT (In die Voice-Liste eintragen)
Syntax: !rmop BENUTZER|*ACCOUNT (Aus der Op-Liste entfernen)
Syntax: !rmvoice BENUTZER|*ACCOUNT (Aus der Voice-Liste entfernen)

Benötigte Rechte: Op

Ändert die Rechte der verwalteten Benutzer.

Synchronisierung

Syntax: !resync

Benötigte Rechte: Op

Gleicht die Rechte Benutzer mit den gespeicherten ab, sodass auch genau die User Op bzw. Voice haben, für die das in der Liste so steht.

(Temporär) Op und Voice vergeben

Syntax: !op BENUTZER (Op geben)
Syntax: !voice BENUTZER (Voice geben)
Syntax: !deop BENUTZER (Op nehmen)
Syntax: !devoice BENUTZER (Op nehmen)

Benötigte Rechte: Op

Gibt einem Benutzer Op bzw. Voice, trägt das aber nicht in die verwaltete Liste ein. D.h. dass die Rechte verwirken, sobald man den Kanal verlässt oder !resync ausgeführt wird.

(Temporär) Sich selbst Rechte geben/nehmen

Syntax: !up (Sich selbst Rechte geben)
Syntax: !down (Sich selbst Rechte geben)

Benötigte Rechte: Keine

Bei !down entzieht ChanServ einem ggf. vorhandene Op- oder Voice-Rechte (wie bei !deop bzw. !devoice).
Bei !up gibt einem ChanServ entsprechend der verwalteten Liste Rechte (wie bei !op bzw. !voice).

Themenverwaltung

Syntax: !set topicmask THEMENMASKE (Themenmaske setzen)
Syntax: !topic THEMA (Thema setzen)

Benötigte Rechte: Op

THEMENMASKE ist dabei ein Text mit einem *, wobei denn ein späterer Aufruf von !topic an Stelle des * das THEMA einfügt und das ganze als Thema setzt.

Beispiel:

!set topicmask [[ Thema: * ]]
!topic Der Himmel ist grün
Das hat dann das Thema [[ Thema: Der Himmel ist grün ]] zur Folge.

Einladung

Syntax: !invite BENUTZER1 BENUTZER2 ...

Benötigte Rechte: Keine

Lädt die hier aufgeführten Benutzer in den Kanal ein.
